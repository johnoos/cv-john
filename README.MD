# Dr John Oosthuizen — Multipage Static Site


This project is a simple static multi-page website ready for GitHub Pages or Netlify. Files to upload are:
- index.html
- about.html
- offerings.html
- contact.html
- assets/style.css
- assets/site.js
- assets/logo.png (add your logo file)
- assets/cv.pdf (add your CV PDF)


## Deploy to GitHub Pages (simple)
1. Create a new **public** repository on GitHub.
2. Upload all files preserving folder structure.
3. In the repository Settings → Pages, set source to `main` branch and `/ (root)` folder.
4. Your site will be available at: `https://YOUR_USERNAME.github.io/REPO_NAME/`
